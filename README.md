# VetApp
приложение для ветклиники(для клиентов)

------------------------------------------------------------------------------------------------

Название: АртВет

Слоган: Лечение -  исскуство, ветеринары - Эскулапы!

------------------------------------------------------------------------------------------------

Функционал приложения:
1) просмотр доступных услуг
2) список докторов и их рейтинг
3) горячая линия\неотложная помощь( с счётчиком спасённых животных) 
etc. 

------------------------------------------------------------------------------------------------

Услуги:

id\название\раздел\цена\доктора(которые выполняют услугу)\вид животных(подходящих под эту услугу)\описание\количество успешных операций\\

------------------------------------------------------------------------------------------------

Доктора:

id\ФИО\раздел\стаж\телефон\кол-во операций(>x)\Рейтинг\количество отзывов

------------------------------------------------------------------------------------------------

виды животных:
1) Кошки
2) Собаки
3) Пернатые
4) Рептилии 
5) Грызуны
6) Экзотические животные
